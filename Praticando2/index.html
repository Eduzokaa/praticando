Itálico e Negrito

Para qualquer frase em itálico, usamos a tag em

E para qualquer frase em negrito, usamos a tag strong

Antes usavamos I para itálico e B para negrito. Aindas são tags que da para usar, mas a qualquer momento podem para de funcionar por serem tags do HTML4

Deixaremos as tags do HTML5 apenas para o significado das coisas e o CSS para a forma das coisas
